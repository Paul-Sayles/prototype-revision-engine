# Prototype Revision Engine

A simple browser-based revision question system for students.

## Purpose
Students should be able to:
- choose a topic
- answer one question at a time
- get immediate feedback
- move through a revision set like slides

## Version 1
- MCQ focus
- simple short-answer/narrative questions at easy levels
- topic-based navigation
- end-of-set score
- reusable across subjects by changing question files
